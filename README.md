# TypeScript Practice Repository

This repository contains **basic TypeScript practice examples** for beginners.  
Each folder has a separate example with **comments explaining TypeScript concepts** and **differences from JavaScript**.

---

## 📂 Folder Structure
```bash
typescript-practice/
 ├─ variable-types/      # Boolean, Number, String
 │    ├─ boolean.ts
 │    ├─ number.ts
 │    └─ string.ts
 ├─ array/               # array practice
 │    └─ array.ts
 ├─ first-code/          # first code example
 │    └─ first-code.ts
 ├─ function/            # function example
 │    └─ function.ts
 ├─ interfaces/          # interfaces example
 │    └─ interfaces.ts
 ├─ object/              # object example
 │    └─ object.ts
 README.md
 tsconfig.json
 ```
